### 1b. Create internal/project/disable.go

Create `/Users/amald/dev/cx/internal/project/disable.go` with package `project`. Implement the following functions exactly as specified in the design:

```
IsDisabled() bool
SetDisabled(reason string) error
ClearDisabled() error
BackupAgentConfig(projectPath, configFile, content string) error
RestoreAgentConfigs() ([]RestoredFile, []string, error)
```

And the type:

```go
type RestoredFile struct {
    ProjectPath string
    ConfigFile  string
}
```

Key implementation details:

- `IsDisabled`: call `GlobalCXDir()`, then `os.Stat(filepath.Join(dir, "disabled"))`. Return true if the file exists (treat any stat error as "not disabled").
- `SetDisabled`: call `GlobalCXDir()`, marshal `{"disabled_at": <RFC3339 now>, "reason": reason}` to JSON, write to `filepath.Join(dir, "disabled")` with mode 0o644.
- `ClearDisabled`: call `GlobalCXDir()`, call `os.Remove(filepath.Join(dir, "disabled"))`. If `os.IsNotExist(err)`, return nil.
- `BackupAgentConfig`: call `GlobalCXDir()`, construct backup dir as `filepath.Join(dir, "agent-backups")`, call `os.MkdirAll` on it, write content to `filepath.Join(backupDir, ProjectID(projectPath)+"-"+configFile)` with mode 0o644.
- `RestoreAgentConfigs`: call `GlobalCXDir()`, read `filepath.Join(dir, "agent-backups")` with `os.ReadDir`. For each entry, parse the filename: the first 12 characters are the project ID, the next character is `-`, and the remainder is the config filename. Call `LoadRegistry()` to get all registered project paths. For each backup, find the matching path by comparing `ProjectID(path)` against the parsed ID. If no match: append a warning (`"warning: no registered project matches backup <filename>; backup retained"`) and continue. If matched: write backup content to `filepath.Join(projectPath, configFile)`, delete the backup file on success, append to the restored list. Return `(restored []RestoredFile, warnings []string, err error)` where `err` is nil unless a fatal I/O failure occurs.

      ```
      <!-- cx disabled — run `cx enable` to restore. Do not edit this file while cx is disabled. -->
      ```
   f. Increment `backedUp`.
5. Call `project.SetDisabled("")`. If error, return it.
6. Print: `fmt.Printf("cx disabled. %d config file(s) backed up.\n", backedUp)`.
7. Return nil.

**Files to modify:**

**Instructions:**

Open `/Users/amald/dev/cx/cmd/root.go`.

2. Add a `PersistentPreRun` field to the `rootCmd` var declaration, inside the `&cobra.Command{...}` literal, after `CompletionOptions`:

```go
PersistentPreRun: func(cmd *cobra.Command, args []string) {
    if !project.IsDisabled() {
        return
    }
    allowed := map[string]bool{"enable": true, "disable": true, "version": true}
    if allowed[cmd.Use] {
        return
    }
    fmt.Fprintln(os.Stderr, "cx is disabled. Run 'cx enable' to restore.")
    os.Exit(1)
},
```

The allowlist uses `cmd.Use` (the leaf command's `Use` field). Cobra passes the leaf command to `PersistentPreRun` so no tree walking is needed. The `version` string matches the auto-generated `--version` subcommand that Cobra creates when `Version` is set.

3. In the `init()` function, append after the existing `AddCommand` calls:

```go
rootCmd.AddCommand(disableCmd)
rootCmd.AddCommand(enableCmd)
```
